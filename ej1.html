¿Qué es un píxel?

Es el elemento mas pequeño de una imagen digital, que cuando se combinan da como resultado una imagen. Todas las pantallas están hechas de píxeles, que pueden mostrar determinados colores gracias a que manejan RGB.
En pantallas de computadores se pueden ver así:

HD 1280px por 720px 921600
Full HD 1920px por 1080px 2073600
4K 3840px por 2160px 8294400

¿Qué es SEO?

Se refiere a Search Engine Optimization, traducido como Optimización de Motores de Busqueda, son las acciones y parametros que se implementan en una página web para mejorar el posicionamiento en los buscadores como Google, Bing, Yahoo, entre otros.
Los métodos mas importantes son:

  • Optimización para moviles.
  • Velocidad del sitio.
  • Inclusión de meta-tags.
  • Inclusión de meta-robots.
  • Inclusión de un mapa del sitio.

¿Qué es CEO?

Se refiere a Chief Executive Officer, que traduce como Director Ejecutivo, es aquella persona encargada de la mas alta responsabilidad en una empresa, ya que de él dependen la mayoría de funciones de la empresa tales como:

  1. Definir la misión, visión y proposito de la empresa.
  2. Tener una visión estratégica de la empresa.
  3. Mantener relaciones con otras empresas aliadas.
  4. Crear un ambiente prospero entre los empleados.
  5. Fijar prioridades a corto, mediano y largo plazo.

¿Qué es Web Semántica?

Son un conjunto de tecnologías creadas para publicar datos legibles para aplicaciones informáticas, con el fin de que los datos sean interoperables sin tener que recurrir a diferentes métodos para obtener datos de otras aplicaciones.
Por ejemplo:

  • Un blog marca las entradas de cine como Cine, al buscar Cine se podrán encontrar todas las entradas con esta marca.
  • Si llega una cita a un correo electrónico, este puede añadirlo a una fecha en el calendario.

¿Qué es Web 2.0?

Con el término Web 2.0, subrayamos un cambio de paradigma sobre la concepción de Internet y sus funcionalidades, que ahora abandonan su marcada unidireccionalidad y se orientan más a facilitar la máxima interacción entre los usuarios y el desarrollo de redes sociales (tecnologías sociales) donde puedan expresarse y opinar, buscar y recibir información de interés, colaborar y crear conocimiento (conocimiento social), compartir contenidos.

¿Qué es Web 3.0?

Esta nueva versión de Internet se encuentra estrechamente ligada con el concepto de 'Web Semántica', el cual, en líneas generales, busca introducir una serie de lenguajes y procedimientos que puedan interpretar ciertas características del usuario con el objetivo de ofrecer una interfaz más personalizada.
